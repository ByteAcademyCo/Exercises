## Problem Description 
AND, OR and NOT are logical operators. What data type is expected for their operands? 

 a) Integer
 b) Boolean
 c) Decimal
 d) Character

Create a variable called `answer` in the **solution.py** and set its value to the string `"a"`, `"b"`, `"c"`, or `"d"`.
