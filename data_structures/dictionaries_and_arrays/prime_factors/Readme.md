# Difference 

## Problem Description 
Define a Python function `prime_factors`, that consumes a positive integer `n`, and returns a list of all the prime factors of `n` in increasing order.

## Examples
```
prime_factors(10) == [1, 2, 5]
prime_factors(13) == [1, 13]
```
