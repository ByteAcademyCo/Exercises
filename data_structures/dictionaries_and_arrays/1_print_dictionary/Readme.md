# print dictionary 

## Problem Description 
Define a python function `print_dict`, that consumes one parameter `dict_num`.The function prints the `values` of the dictionary `dict_num`,one on each line.

## Examples
```
dict_num = ({1: 10, 2: 20, 3: 30})
print_dict(dict_num)
Output 
10
20
30
```
